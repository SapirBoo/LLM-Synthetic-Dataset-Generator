# LLM-Synthetic-Dataset-Generator
Synthetic dataset generator that supports multiple instruction-tuned models.

Generate structured synthetic datasets using local Hugging Face models.

## Features

- Gradio UI
- Multiple local LLMs
- Qwen2.5-3B
- Phi-3.5 Mini
- SmolLM2
- TinyLlama
- 4-bit Quantization (BitsAndBytes)
- Automatic JSON parsing
- CSV export

## Technologies

- Python
- Hugging Face Transformers
- BitsAndBytes
- Gradio
- Pandas
- PyTorch

## Example
Generate datasets such as:

- Yoga Studio
- Dance Studio
- Gym
- Martial Arts Studio

Output:

- Interactive table
- CSV download
